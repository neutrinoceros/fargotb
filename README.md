### Instalation

recommanded method : 
copy the source in the directory of your choice
```
git clone https://github.com/neutrinoceros/fargotb.git
```

then

```
cd fargotb
./install.sh
```

the script makes symbolic links for each executable file and put them into your ~/bin/ directory.
